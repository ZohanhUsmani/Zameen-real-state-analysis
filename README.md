## Overview
This project analyzes real estate property listings from Zameen.com to study property availability and pricing trends across major cities in Pakistan.

The analysis focuses on different property types such as Flats, Houses, Penthouses, Upper Portions, and Lower Portions, using data visualization and statistical techniques to identify market patterns.

## Business Objective
To evaluate property availability and pricing trends to support informed decision-making for:
- Investors
- Real estate developers
- Property buyers

## Dataset
- Source: Zameen.com
- Format: Cleaned CSV file
- Features:
  - City
  - Property Type
  - Price (Million PKR)
  - Listing Count (derived)
  - Average Price (derived)

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Jupyter Notebook / Python scripts

## Analysis Performed
- Property type distribution
- City-wise availability analysis (heatmaps)
- Average price comparison by city and property type
- Price vs availability relationship (bubble chart)
